## 오늘 한 것

### P1 프로젝트 초기 구축 (백엔드 + 프론트엔드 파일 추가)

**백엔드 (NestJS + TypeORM)**
- 엔티티 설계 및 생성
  - `User`, `Course`, `CourseSection`, `CourseVideo`
  - `Enrollment`, `VideoProgress`
  - `CartItem`, `Question`, `Answer`
- 모듈 및 API 구현
  - `Auth` (로그인/회원가입)
  - `Courses` (강의 목록·상세 조회)
  - `Cart` (장바구니 담기·삭제)
  - `Enrollments` (수강신청·진도 저장)
  - `Questions` (Q&A 질문·답변)
- 시드 데이터 작성 (`src/seed.ts`)

**프론트엔드 (React + Vite + TSX)**
- 전체 페이지 구현 (TSX 기반)
  - 강의 목록(`CourseList`), 강의 상세(`CourseDetail`)
  - 로그인(`Login`), 회원가입(`Signup`)
  - 장바구니(`Cart`), 결제완료(`CheckoutComplete`)
  - 내 강의실(`Dashboard`), 수업 듣기(`VideoPlayerPage`)
  - Q&A (`QuestionDetail`, `QuestionNew`, `QuestionEdit`)
- 공용 컴포넌트 구현
  - `Button`, `Modal`, `Tabs`, `Accordion`
  - `FilterSidebar`, `CourseCard`, `VideoPlayer`
  - `Pagination`, `ProgressBar`, `EmptyState`
- Redux 슬라이스 구성 (`user`, `courses`, `cart`, `enrollment`, `player`, `qa`)
- API 서비스 레이어 (`api.js`), 유틸 (`format.js`, `constants.js`)
