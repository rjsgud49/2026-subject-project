# 프로젝트 오버뷰 (P1)

## 1. 프로젝트명·주제

- **프로젝트명**: 면접 인강 플랫폼 (P1)
- **주제**: 면접(인터뷰 준비) 온라인 강의 플랫폼의 고객(학생) 기능 구현

---

## 2. 목적

- 취업·이직 준비생이 **면접 강의를 검색**하고, **수강신청**한 뒤, **내 수업**에서 **영상을 시청**하며 **질문(Q&A)**을 할 수 있는 최소 기능을 P1에서 구현합니다.

---

## 3. P1 범위 (요약)

| 영역 | 내용 |
|------|------|
| **검색** | 키워드, 카테고리, 난이도, 가격대, 정렬, 페이징 |
| **수강신청** | 강의 상세, 장바구니, 간이 결제, 수강 등록 |
| **내 수업** | 수강 목록, 진도율, 수료 여부, 이어보기 |
| **수업 듣기** | 영상 재생, 진도 저장, 목차 이동, 완료 처리 |
| **질문하기** | 강의별 Q&A 목록/상세, 질문 등록/수정/삭제, 답변 등록 |

---

## 4. 기술 스택 (가이드)

- **백엔드**: REST API (언어·프레임워크 자유, OpenAPI 스펙 준수 권장)
- **프론트엔드**: SPA 또는 SSR (React, Vue, Next 등 자유)
- **DB**: RDB (MySQL, PostgreSQL 등), ERD 기준 테이블
- **영상**: P1에서는 URL 기반 재생(파일 URL 또는 스트리밍 URL)
