# 2026-04-17 작업 일지

## 오늘 한 것

**P2 LMS (강사·피드백·정산·저장소)**

- **강사 스튜디오 UI**: `TeacherWorkspace` 공통 레이아웃, 대시보드·내 강의·강의 폼·피드백·프로필 화면 정리 및 로딩·에러 처리 보강
- **강사 대시보드**: `GET /teacher/dashboard` 연동 — 강의별 조회수·수강 수·매출·플랫폼 수수료(10%)·예상 정산 표시
- **강의**: `Course.view_count` 및 공개 상세 조회 시 증가, 커리큘럼은 JSON 대신 `CurriculumEditor` UI로 편집, 강의 영상 업로드는 **영상 파일만** 허용(`upload.validation`)
- **피드백**: 학생 첨부용 `POST /feedback/upload`, DB `student_attachments_json`, 강사 화면에 첨부 링크·무첨부 안내, BigInt `id` 비교로 수정 화면 “내 강의 아님” 오류 완화
- **정산 계좌**: `User`에 은행·계좌·예금주 필드, `PATCH /teacher/profile`·`/auth/me` 반영, `/teacher/settlement` 페이지 및 내비 추가
- **업로드 저장소**: 기본 `backend/uploads/p2`, 환경변수 `P2_UPLOAD_DIR`로 로컬 경로 지정 가능, `.gitignore`에 `p2/backend/uploads/` 제외
- **강사 시점 UI 분기**: `MainLayout`에서 장바구니·학생용 피드백 드롭다운 숨김, `CourseDetail`·`Cart`에서 수강/장바구니 비활성, `/feedback/*`는 강사·관리자 리다이렉트 훅 적용

## 오류 / 어려웠던 점

- 프론트에서 API `id`가 문자열로 올 때 `===` 비교로 강의 매칭이 실패하는 경우가 있어, 비교 시 `Number()` 정규화 필요
- 첨부가 없는 기존 피드백 건은 DB에 첨부 컬럼이 비어 있어 UI에 링크가 안 나오는 것이 정상 — 사용자에게 구역·안내 문구로 가시성 확보
