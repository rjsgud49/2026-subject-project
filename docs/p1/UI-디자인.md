# UI 디자인 (P1)

## 1. 개요

- 고객(학생) 기준 화면·플로우를 정의합니다.
- P1에서는 와이어프레임 수준의 구성과 플로우만 정리합니다.

---

## 2. 공통 레이아웃

- **상단**: 로고, 검색창, 메뉴(강의 탐색, 내 수업, 장바구니)
- **본문**: 각 페이지별 콘텐츠
- **하단**: 푸터(선택)

---

## 3. 화면별 구성

### 3.1 강의 검색·목록

- **검색창**: 상단 또는 전용 검색 페이지
- **필터**: 카테고리(칩/버튼), 난이도, 가격대(슬라이더 또는 입력)
- **정렬**: 드롭다운(최신순, 가격 낮은순/높은순 등)
- **결과**: 카드 그리드 — 썸네일, 제목, 강사명, 가격, 평점(선택), "상세 보기"

### 3.2 강의 상세

- **상단**: 썸네일, 제목, 강사명, 가격, "장바구니 담기" / "수강신청" 버튼
- **탭**: 소개 | 커리큘럼 | Q&A
  - **소개**: 강의 설명, 강사 소개
  - **커리큘럼**: 섹션별 영상 목록(제목, 재생 시간)
  - **Q&A**: 질문 목록 + "질문하기" 버튼

### 3.3 장바구니

- **목록**: 강의 카드(썸네일, 제목, 가격, 삭제 버튼)
- **합계**: 총 금액
- **버튼**: "결제하기" → 결제 완료 페이지(간이) → "내 수업 보기"

### 3.4 내 수업

- **목록**: 수강 중인 강의 카드
  - 썸네일, 제목, 진도율(프로그레스 바), "수강 중" / "수료" 뱃지
  - "이어보기" 또는 "들어가기" 버튼
- **필터**: 수강 중 / 수료 완료 (탭 또는 필터)

### 3.5 수업 듣기

- **좌측/상단**: 비디오 플레이어
  - 재생/일시정지, 볼륨, 전체화면, 재생 속도, 구간 반복(선택)
- **우측/하단**: 목차
  - 섹션 > 영상 제목 목록, 클릭 시 해당 영상 재생, 현재 재생 영상 하이라이트
- **탭**: 재생 화면 내 또는 상단에 "Q&A" 링크 → 강의 Q&A로 이동

### 3.6 질문하기 (Q&A)

- **목록**: 제목, 작성자, 날짜, 답변 수 — "질문하기" 버튼
- **질문 상세**: 제목, 본문, 작성자, 날짜, 답변 목록, "답변 작성"(권한에 따라 P2)
- **질문 작성**: 모달 또는 전용 페이지 — 제목, 본문 입력 후 등록

---

## 4. 주요 플로우

1. **검색 → 상세 → 장바구니 → 결제 → 내 수업**
2. **내 수업 → 이어보기 → 수업 듣기(진도 자동 저장)**
3. **강의 상세 또는 수업 듣기 → Q&A 탭 → 질문 목록 → 질문 작성/상세**

---

## 5. 반응형

- P1에서는 **데스크톱 웹** 우선.
- 카드 그리드는 3~4열, 필요 시 2열·1열로 줄이는 정도만 고려해도 됨.
