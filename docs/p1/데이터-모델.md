# 데이터 모델 (P1)

## 1. 개요

- ERD(`erd.md`)의 테이블을 도메인 관점에서 정리합니다.
- P1에서는 고객(학생) 기능에 필요한 엔티티만 사용합니다.

---

## 2. 핵심 도메인

### 2.1 사용자 (User)

- **역할**: 수강생(학생), 강사, 관리자 — P2에서 역할별 제어
- **P1에서 사용**: 수강신청·장바구니·진도·Q&A의 주체 식별
- **주요 필드**: id, email, name, (password_hash, role은 P2에서 활용)

---

### 2.2 강의 (Course)

- **의미**: 면접 강의 1개 (제목, 설명, 강사, 카테고리, 난이도, 가격)
- **관계**: 강사(users), 섹션(course_sections) → 영상(course_videos)
- **상태**: is_published로 공개/비공개 (목록에는 공개만 노출)

---

### 2.3 강의 섹션 (CourseSection)

- **의미**: 강의 내 목차 단위 (예: "1장. 자기소개", "2장. 기술 질문")
- **관계**: course 1 : N sections, section 1 : N videos
- **정렬**: sort_order로 목차 순서 보장

---

### 2.4 강의 영상 (CourseVideo)

- **의미**: 실제 시청 단위 영상 (URL, 재생시간)
- **관계**: section N : 1, video_progress와 연결해 진도 저장
- **진도 계산**: 완료된 영상 수 / 전체 영상 수 → 진도율 %

---

### 2.5 수강 (Enrollment)

- **의미**: 사용자와 강의의 수강 관계 1건
- **제약**: (user_id, course_id) 유일 → 중복 수강 방지
- **상태**: active(수강 중), completed(수료)
- **수료 조건**: 해당 강의의 모든 영상이 completed이면 enrollment.status = completed

---

### 2.6 영상 진도 (VideoProgress)

- **의미**: 수강 1건에 대해 영상별로 “어디까지 봤는지”, “완료 여부”
- **제약**: (enrollment_id, video_id) 유일
- **필드**: last_second(마지막 시청 초), completed(해당 영상 완료 여부)
- **완료 정책**: 시청한 길이가 duration_seconds의 90% 이상이면 completed = true (정책 변경 가능)

---

### 2.7 장바구니 (CartItem)

- **의미**: 결제 전 “담기”한 강의
- **제약**: (user_id, course_id) 유일, 이미 수강 중이면 담기 불가
- **삭제**: 결제 완료 후 삭제 또는 사용자가 직접 삭제

---

### 2.8 질문·답변 (Question, Answer)

- **Question**: 강의별 Q&A 게시글 (제목, 본문, 작성자, 강의 ID)
- **Answer**: 질문에 대한 답변 (본문, 작성자) — P2에서 강사/학생 구분 가능
- **관계**: course 1 : N questions, question 1 : N answers

---

## 3. 파생/계산 값

| 값 | 설명 | 계산 방식 |
|----|------|-----------|
| 진도율 | 강의별 진행 % | (completed인 video 수 / 해당 강의 전체 video 수) × 100 |
| 수료 여부 | enrollment.status | 모든 영상 completed 시 completed |
| 이어보기 | 마지막 시청 영상·초 | video_progress에서 해당 enrollment의 max(updated_at) 행의 video_id, last_second |

---

## 4. 인덱스 요약

- **검색**: courses(category, difficulty, price, is_published, created_at)
- **내 수업**: enrollments(user_id), video_progress(enrollment_id, video_id)
- **Q&A**: questions(course_id, created_at)
- **장바구니**: cart_items(user_id)

상세 컬럼은 `erd.md` 참고.
